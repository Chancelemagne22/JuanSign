│ DESIGN PATTERNS USED                                                        │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  1️⃣  FLUID TYPOGRAPHY (clamp)                                              │
│     Smoothly scales text between min and max based on viewport            │
│     Example: clamp(2.5rem, 8vw, 5rem)                                    │
│                                                                             │
│  2️⃣  RESPONSIVE SPACING (clamp)                                            │
│     Automatically adjust padding/gaps with screen size                    │
│     Example: clamp(1rem, 4vw, 1.5rem)                                   │
│                                                                             │
│  3️⃣  SAFE CONTAINER WIDTHS (min)                                           │
│     Takes full width with gutters, capped at design max                   │
│     Example: max-width: min(100% - 2rem, 500px)                         │
│                                                                             │
│  4️⃣  LAYOUT BREAKPOINTS (Tailwind)                                         │
│     Changes layout at specific screen sizes                               │
│     Example: flex flex-col md:flex-row                                   │
│                                                                             │
│  5️⃣  MEDIA QUERIES (CSS)                                                    │
│     Different styling below/above breakpoint                              │
│     Example: @media (max-width: 767px) { ... }                           │
│                                                                             │
